# 광산구 선거일투표소 지도

광산구 선거일투표소 95곳을 네이버지도에 표시하고, 투표소별 반경 100m를 확인하거나 A4 가로 용지로 인쇄할 수 있는 지도입니다.

공개 주소: https://rmsummit.github.io/gwangsan-polling-map/

## 주요 기능

- 광산구 전체 선거일투표소 95곳 지도 표시
- 선관위 약칭 `제1투` 형식은 화면에서 `제1투표소`로 풀어서 표시
- 투표소 마커를 빨간 선거도장 모양으로 표시
- 지도 확대 단계에 따라 투표소 마커 크기 자동 조정
- 모바일/데스크톱 공통 지도유형 전환 버튼
- 광산구 갑/을 전체 필터
- 가/나/다/라/마 선거구별 필터
- 선거구 안의 특정 동 필터
- 투표소명, 동, 주소 검색
- 투표소별 반경 100m 원 표시
- 현재 내 위치 표시
- 내 위치 기준 가장 가까운 투표소와의 100m 이내/밖 상태 표시
- 모바일 지도 위 고정 `내 위치` 재조회 버튼
- 선택한 투표소 A4 가로 인쇄
- 카카오톡 공유용 링크 복사 및 모바일 공유

## 사용 방법

1. 지도 페이지를 엽니다.
   - https://rmsummit.github.io/gwangsan-polling-map/

2. 선거구를 선택합니다.
   - `전체 선거구`: 모든 투표소를 봅니다.
   - `광산구 갑 전체`, `광산구 을 전체`: 갑/을 구역 전체를 봅니다.
   - `가선거구`, `나선거구`, `다선거구`, `라선거구`, `마선거구`: 세부 선거구만 봅니다.

3. 필요한 경우 동을 선택합니다.
   - 선거구를 선택하면 해당 선거구에 속한 동만 `동 선택` 목록에 표시됩니다.
   - `동 전체`를 선택하면 선택한 선거구의 모든 동을 봅니다.

4. 검색창을 사용합니다.
   - 동 이름, 투표소명, 주소 일부를 입력하면 현재 선택된 선거구/동 안에서 다시 검색됩니다.
   - 예: `첨단`, `운남`, `초등학교`, `하남대로`

5. 투표소를 선택합니다.
   - 지도 마커 또는 왼쪽 목록의 투표소를 누르면 해당 투표소만 표시됩니다.
   - 상단에는 선택한 투표소명과 주소가 표시됩니다.

6. 다시 필터 결과로 돌아갑니다.
   - 특정 투표소만 보이는 상태에서 `선택 해제`를 누르면 현재 선택한 선거구/동/검색 결과 전체로 돌아갑니다.

7. 전체 보기로 초기화합니다.
   - `전체 보기`를 누르면 선택한 선거구/동은 유지한 채 검색어와 선택 투표소만 초기화됩니다.
   - 선거구가 선택된 상태라면 해당 선거구의 투표소 전체로 돌아갑니다.

8. 인쇄합니다.
   - 투표소를 선택한 뒤 `A4 인쇄`를 누르면 해당 투표소 기준 반경 100m 지도를 A4 가로 방향으로 인쇄할 수 있습니다.
   - 투표소를 선택하지 않고 인쇄하면 현재 필터 결과의 첫 번째 투표소가 인쇄 대상으로 선택됩니다.

9. 지도 유형을 바꿉니다.
   - 지도 오른쪽 위의 `일반` 버튼을 누르면 `일반`/`위성` 지도를 전환할 수 있습니다.
   - 모바일과 데스크톱 모두 같은 위치에 표시됩니다.

10. 현재 위치를 표시합니다.
   - 상단의 `내 위치`를 누릅니다.
   - 브라우저 위치 권한을 허용하면 지도에 작은 빨간 점으로 현재 위치가 표시됩니다.
   - 내 위치에서 가장 가까운 투표소가 100m 안이면 `경고`, 100m 밖이면 `안전` 상태가 표시됩니다.
   - 메뉴가 접힌 뒤에도 지도 위의 `내 위치` 버튼으로 다시 확인할 수 있습니다.
   - 위치 기능은 HTTPS 공개 주소에서 정상 동작합니다. `file://`로 열면 제한될 수 있습니다.

11. 카카오톡으로 공유합니다.
   - 상단의 `카카오톡 공유`를 누릅니다.
   - 모바일 브라우저에서는 공유창에서 카카오톡을 선택할 수 있습니다.
   - PC나 공유창을 지원하지 않는 브라우저에서는 공개 주소가 복사됩니다. 카카오톡 대화창에 붙여넣으면 됩니다.
   - 카카오톡 링크 미리보기에는 `share-infographic.png` 이미지가 표시됩니다.

## 모바일 사용 팁

- 모바일에서는 지도를 먼저 볼 수 있도록 조회 메뉴가 접힌 상태로 시작합니다.
- `조회 열기`를 누르면 선거구, 동, 검색, 인쇄, 공유 버튼이 펼쳐집니다.
- 메뉴가 열린 상태에서 `지도 보기`를 누르면 다시 접힙니다.
- 선거구를 선택하면 동까지 이어서 고를 수 있도록 메뉴가 열린 상태로 유지됩니다.
- 동을 선택하거나, 투표소/내 위치로 이동하면 메뉴가 자동으로 접혀 지도를 넓게 볼 수 있습니다.
- 메뉴가 접힌 상태에서도 지도 위의 `내 위치` 버튼으로 현재 위치를 다시 확인할 수 있습니다.

## 선거구별 동

| 구분 | 동 |
| --- | --- |
| 광산구 갑 전체 | 가선거구 + 나선거구 |
| 광산구 을 전체 | 다선거구 + 라선거구 + 마선거구 |
| 가선거구 | 송정1동, 송정2동, 도산동, 어룡동, 동곡동, 평동, 삼도동, 본량동 |
| 나선거구 | 신흥동, 우산동, 월곡1동, 월곡2동, 운남동 |
| 다선거구 | 첨단1동, 첨단2동 |
| 라선거구 | 신가동, 신창동, 비아동 |
| 마선거구 | 수완동, 하남동, 임곡동 |

| 일반/위성 | 지도 오른쪽 위에서 기본 지도와 위성 지도를 전환합니다. |
| 내 위치 | 현재 위치를 작은 빨간 점으로 표시하고, 가장 가까운 투표소 기준 100m 이내/밖 상태를 표시합니다. |
| 카카오톡 공유 | 모바일 공유창을 열거나 공개 주소를 복사합니다. |
| A4 인쇄 | 선택한 투표소의 반경 100m 지도를 인쇄합니다. |
| 선택 해제 | 선택한 투표소 1곳 보기에서 현재 필터 결과로 돌아갑니다. |
| 전체 보기 | 선택한 선거구/동은 유지하고 검색어와 선택 투표소만 초기화합니다. |
| 팝업 닫기 | 지도 위 투표소 정보 팝업만 닫습니다. |

## 자주 묻는 문제

### 지도가 안 보이고 "네이버 지도 Open API 인증이 실패했습니다"라고 나옵니다.

`index.html` 파일을 더블클릭해서 `file://` 주소로 열면 네이버 지도 API 인증이 실패합니다. 반드시 GitHub Pages 주소로 접속하세요.

정상 주소:

```text
https://rmsummit.github.io/gwangsan-polling-map/
```

### 수정했는데 사이트에 바로 안 보입니다.

GitHub Pages 배포에는 보통 수십 초 정도 걸립니다. 배포 후에도 예전 화면이 보이면 브라우저에서 강력 새로고침을 해주세요.

```text
Mac: Cmd + Shift + R
Windows: Ctrl + Shift + R
```

## 배포 전 점검

수정 후 배포하기 전에 아래 명령으로 주요 회귀를 확인합니다.

```bash
node regression-check.js
```

현재 점검 항목:

- `전체 보기`가 선택한 선거구/동 필터를 유지하는지 확인
- README 사용법이 실제 동작과 일치하는지 확인
- 투표소 좌표 보정 테이블과 좌표 해석 함수가 유지되는지 확인

좌표 검수용 CSV는 아래 명령으로 생성합니다.

```bash
node coordinate-audit.js
```

좌표가 어긋난 투표소는 `index.html`의 `coordinateOverrides`에 보정 좌표를 추가합니다. 보정 좌표는 마커, 반경 100m 원, A4 인쇄 중심, 내 위치 100m 판정에 함께 적용됩니다.

## 포함 파일

- `index.html`: 배포용 지도 페이지
- `README.md`: 사용법 및 배포 안내
- `regression-check.js`: 배포 전 회귀 점검 스크립트
- `coordinate-audit.js`: 학교/대형부지 좌표 검수 CSV 생성 스크립트
- `coordinate-audit.csv`: 좌표 검수 대상 목록
- `polling-stamp.svg`: 투표소 마커 아이콘
- `share-infographic.svg`: 카카오톡 미리보기용 인포그래픽 원본
- `share-infographic.png`: 카카오톡 미리보기용 인포그래픽 이미지
- `gwangsan_polling_places_with_naver_coordinates.xlsx`: 투표소 주소와 네이버 지오코딩 좌표
